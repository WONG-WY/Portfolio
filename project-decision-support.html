Project

Full-Stack School Decision-Support Tool

Real-time school comparison platform for smarter decisions

Vue.js Node.js Auth0 MongoDB Express.js

Overview

A full-stack web platform that helps users compare schools using live government data. Built with a custom authentication layer, real-time API integration, and a resilient data-sync architecture to ensure accuracy and reliability.

What Was Built

  • Built a Vue.js and Node.js platform integrating government APIs for real-time school data comparison
  • Implemented Auth0 Headless Authentication (ROPC) to secure APIs with JWTs while retaining a custom UI
  • Engineered a dual-write mechanism via Express.js to sync MongoDB as the single Source of Truth

Key Outcomes

  • Secure, token-authenticated API layer with no compromise on custom UI design
  • Reliable data consistency between external APIs and local database via dual-write sync
  • End-to-end full-stack ownership from frontend components to backend data pipelines
