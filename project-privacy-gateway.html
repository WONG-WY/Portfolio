Project

Privacy Gateway on Microsoft Azure

IP masking & ad-blocking gateway

Azure VM OpenVPN Pi-hole Linux

Overview

Deployed a full privacy gateway on Microsoft Azure to provide encrypted browsing and network-level ad blocking. The system combines a VPN server with a DNS sinkhole, giving users a private, ad-free internet experience routed through a cloud VM.

What Was Built

  • Deployed an OpenVPN server on an Azure VM to establish encrypted tunnels and mask client IP addresses
  • Integrated a Pi-hole DNS sinkhole with a ~180,000-domain blocklist to block ads and minimize latency
  • Validated IP masking and ad-blocking effectiveness through end-to-end testing with third-party tools

Key Outcomes

  • Full IP anonymisation confirmed via external IP-check services
  • Significant reduction in ad-related DNS queries across connected devices
  • Demonstrated practical cloud infrastructure and network security skills
